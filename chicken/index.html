مدیریت مزارع پرورش گوشت

تجاری

مدیریت و حسابرسی آسان مزارع پرورش گوشت بزرگ و متوسط و حتی کوچک

امکانات

  • حسابداری
  • انبارداری
  • پنل مدیریت مجموعه
  • پنل گزارشات سالنها
  • قابلیت آنلاین با کاربر بینهایت
  • اپلیکیشن موبایل

همکاران

مشاور پروژه:
مهندس عبدالحکیم نعمایی (مدیر مالی)
مهندس عبدالواحد دلشاد (حسابدار ارشد)

تکنولوژی‌ها

  • B4J (desktop dashboard)
  • JavaFX (desktop ui)
  • B4A (mobile dashboard)
  • SQLite
  • B4J (back-end)
  • MySql
  • Socket (communication protocol)
  • Java

وظایف من در پروژه

  • توسعه طرح رابط کاربری و تجربه راحت مدیریت برای کاربران
  • توسعه برنامه پایه بر مبنای Java و JavaFX
  • طراحی کل پروسه مدیریت مزارع با تعداد بالای سالن با راحترین ورش ممکن
  • طراحی و توسعه بخش حسابداری به روشی نوین و کارا (با همکاری و نظرات حسابداران و حساب رسان)
  • طراحی و توسعه دیتابیس با MySql
  • پیاده سازی شبکه ای برای برقراری ارتباط کارا بین پرسنل با Socket
  • توسعه اپ اندروید برای مدیریت بخش‌های مختلف مدیریتی، حسابدرای
  • قابلیت اضافه کردن پلتفورم میراب

تکمیلی

  • توانای اصلی برنامه این است که پیچیدگی مدیریت و حاسبداری سیستم بزرگ مزرعه‌های تولید گوشت (مرغداری) به آسانترین روش ممکن و با مشارکت همزمان تمام پرسنل رفع شود.
